Druxt.js Blocks

Provides Drupal blocks and region components to be used within a Druxt (DRUpal nuXT) project.

Get started

The Druxt.js Blocks module adds Drupal Blocks, Content Blocks and Block Region Vue.js components to your Nuxt.js frontend.

Example: Header region for Umami theme.

<druxt-block-region
  name="header"
  theme="umami"
/>

Example: Block Content block rendered by UUID.

<druxt-block
  uuid="baefa4d3-9517-4413-8b9e-975c8affb8ac"
/>

# Druxt.js

Druxt.js is a suite of modules to connect a Drupal JSON:API backend to a Nuxt.js frontend.

Find out more at http://druxtjs.org
